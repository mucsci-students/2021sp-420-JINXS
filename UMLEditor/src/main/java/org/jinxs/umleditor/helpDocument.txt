Command/shortcut	          Arguments		                                                                  Description
-------------------------	  --------------	                                                              --------------
addClass                          name of class                                                               Creates a new UML class 

deleteClass                       name of existing class                                                      Deletes the specified class

renameClass                       name of existing class, new name                                            Renames a class to a new specified class name                                                                                                                                 
                                                                                                                                                       
addRel                            name of existing class (src), 
                                  name of 2nd existing class (dest)                                           Adds a relationship between the two given classes                                                                      

deleteRel                         name of existing class (src),
                                  name of 2nd existing class (dest)                                           Deletes a relationship between the two given classes 

addAttr                           name of existing class, attribute name                                      Adds an attribute to a given class 

deleteAttr                        name of existing class, attribute name                                      Deletes the given attribute from the specified class

renameAttr                        name of existing class, name of existing attribute,                         Renames the given specified attribute in the given                                                                             new name for attribute                                                                                          class to a new name for the attribute

printList                         none                                                                        Prints the names of all existing classes

printContents                     name of existing class                                                      Prints the contents of a given class

printRel                          name of existing class, name of 2nd existing class                          Prints the all the relationships between classes 

help                              none                                                                        Prints a help document with all viable commands 

quit                              none                                                                        Exits the the program 

save                              name of JSON file                                                           Saves the project into a JSON file

load                              name of JSON file                                                           Loads a project from a JSON file
